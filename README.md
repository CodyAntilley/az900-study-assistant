# AZ-900 Study Assistant

A structured, easy-to-understand collection of study notes for the Microsoft Azure Fundamentals (AZ-900) exam.  
This repository is designed to support a custom AI-powered study bot built using Azure AI Studio.

## 📘 What This Repo Contains
- Clean, original summaries of every AZ-900 topic
- Organized by module for easy navigation
- Simple explanations for simple topics
- Deeper breakdowns for complex topics
- Perfect for training an Azure AI chatbot or personal study assistant

## 📂 Folder Structure
- **module-01-cloud-concepts** — Cloud basics, service models, deployment models
- **module-02-azure-architecture** — Compute, storage, networking, databases
- **module-03-security-identity-governance** — Identity, access, security, governance
- **module-04-monitoring-and-management** — Monitoring tools and service health
- **module-05-cost-management** — Pricing, TCO, budgets, cost optimization

## 🎯 Purpose
This repo supports:
- Azure AI Studio custom chatbots
- Personal study workflows
- Portfolio projects demonstrating cloud knowledge
- Anyone preparing for the AZ-900 exam

## 🚀 How to Use
1. Clone or download the repo  
2. Upload the `.txt` files into Azure AI Studio as knowledge sources  
3. Build your study bot  
4. Ask questions and reinforce your learning
